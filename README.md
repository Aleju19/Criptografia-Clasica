# 🔐 Sistema de Criptografía Clásica
Este proyecto es una aplicación de escritorio desarrollada en **Java 21/25** utilizando **JavaFX** y el sistema de 
construcción **Gradle**. Implementa los algoritmos de cifrado más emblemáticos de la historia bajo una arquitectura sólida basada en el **Patrón Factory**.

## 🚀 Algoritmos Implementados
* **César:** Sustitución por desplazamiento.
* **Atbash:** Sustitución recíproca.
* **Vigenère:** Cifrado polialfabético mediante palabra clave.
* **Rail Fence:** Transposición en zigzag.
* **Playfair:** Sustitución de digramas en matriz 5x5.

## 🛠️ Tecnologías
* **Lenguaje:** Java 
* **Framework UI:** JavaFX
* **Gestor de Dependencias:** Gradle
* **Control de Versiones:** Git / GitHub

## 📝 Instrucciones
Usar el jdk 21
1. Clonar el repositorio.
2. Ejecutar a través de la clase `Launcher`.
